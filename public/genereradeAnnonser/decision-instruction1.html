Sprinta söker Frontend- / Backend-utvecklare

Om Företaget

Sprinta är en dynamisk tech-startup som utvecklar en applikation för AI-genererade jobbannonser. Vi kombinerar React, Java, Azure och AI-teknik för att skapa vår banbrytande produkt. Vi är ett dedikerat och växande team som nu letar efter en extra utvecklare för att snabba på vår utveckling och lansering.

Jobbroll

Vi söker en passionerad och skicklig utvecklare som kan bidra till vårt frontend och/eller backend. Rollen innebär att du kommer att arbeta med våra nuvarande utvecklare och bli en viktig del i lanseringen av vår applikation.

Ansvarsområden

  • Utveckla och underhålla vår frontend i React, eller vår backend i Java.
  • Jobba med molntjänsten Azure.
  • Effektiv användning av GIT och Github som versionshanterare.
  • Flexibilitet att lära och anpassa sig till nya bibliotek och ramverk.

Kvalifikationer

  • Starka kunskaper i React och dess bibliotek, eller Java och Spring för backend utveckling.
  • Erarenhet med arbete i Azure-miljö.
  • Effektiv kunnighet av GIT och Github.
  • Utmärkt problemlösningsförmåga och vilja att ta initiativ.
  • Förmåga att arbeta som en del av ett team.
