Frontend-/Backendutvecklare för AI Genererade Jobbannonser

Vi utvecklar en revolutionerande applikation för AI-genererade jobbannonser och letar nu efter en begåvad och passionerad utvecklare för att utveckla vår snabbt växande applikation. Vi bygger något stort - vill du vara med oss på resan?

Om tjänsten

Du kommer att arbeta med vår Frontend skriven i React, använda libraries som React Router, DOMPurify, CSS och Styled Components. På Backend-sidan har vi skrivit koden i Java och Spring Boot, med Spring Security samt kommunikation med ett AI API. Databasen sköts av MySQL och allt styrs genom molntjänsten Azure. Vi arbetar med tiha GIT och GitHub för versionshantering.

Vi räknar med att sjösätta applikationen inom 1 år, och som en del av vårt team kommer du att ha en stark inverkan på dess framgång.

Dina kvalifikationer

Vi söker dig som älskar att lära dig nya koncept och tekniker. Du behöver vara beredd att ständigt anpassa dig och lära dig nya libraries och frameworks för att hålla jämna steg med vår snabbt föränderliga tekniska miljö. Du bör ha erfarenhet av Front- och Backend-utveckling, inklusive bekantskap med alla de tekniker som nämnts ovan.
