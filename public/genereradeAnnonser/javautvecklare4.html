Jobbets innehåll

Som Fullstack Utvecklare kommer du att ha en avgörande roll i utvecklingen av vår AI-genererade jobbannonseringsapp. Din dag kommer att innefatta:

  • Utveckling på både frontend i React, med libraries som React Router, Styled Components, Bootstrap och DOMPurify,
  • och backend i Java, med Spring Boot och Spring Security,
  • Samt kommunikation med vårt AI API.

Vi använder MySQL för databashantering och Azure för våra molntjänster. Versionskontroll sker genom Git och Github. Nyckeln till vår framgång ligger i ständigt lärande och anpassning, och vi uppskattar teammedlemmar som är öppna för att utforska nya bibliotek och ramar.
