Fullstack Utvecklare på DevSop

Är du en passionerad utvecklare som brinner för både frontend och backend-utveckling? Vill du vara en del av ett dynamiskt team där du får chansen att arbeta med moderna teknologier och verktyg? Då kan tjänsten som Fullstack Utvecklare hos oss på DevSop vara något för dig!

Om Oss

DevSop är ett framstående tech-företag som specialiserar sig på att skapa innovativa lösningar för våra kunder. Vi värdesätter kreativitet, samarbete och en drivkraft att ständigt förbättras.

Om Rollen

I rollen som Fullstack Utvecklare kommer du att arbeta med att utveckla en interaktiv web applikation med hjälp av React för frontend, Spring Boot för backend och hantera databasen med MySQL. Du kommer att få möjlighet att arbeta både självständigt och i team, och vi uppmuntrar till att ständigt utforska och införliva nya teknologier och verktyg i vårt arbete.

Teknologier Och Verktyg

  • Frontend: React, React Router, DOMPurify, Bootstrap 5, CSS, Styled Components
  • Backend: Java, Spring Boot, Spring Security, AI API
  • Databas: MySQL
  • Molnplattform: Azure

Kvalifikationer

  • Erfarenhet av att arbeta med både frontend och backend utveckling.
  • Goda kunskaper i React, Java och MySQL.
  • Erfarenhet av att arbeta med Spring Boot och Spring Security.
  • Möjlighet att arbeta både självständigt och i team.

Vad Vi Erbjuder

  • En dynamisk och inspirerande arbetsmiljö.
  • Stora möjligheter till personlig och professionell utveckling.
  • Flexibla arbetstider och möjlighet att arbeta på distans.
  • Generösa förmåner och konkurrenskraftig lön.
